About the measurement

HTTP-based quality estimates for everyday gaming readiness

Ping Test: Gaming Speed estimates network quality using HTTP-based measurements. Results help you understand current connection conditions before playing, including response time, stability, download performance, failed responses, and an overall gaming score.

The app does not claim to measure real game server latency. Actual in-game latency can differ depending on the game, region, server routing, platform, and network congestion.
