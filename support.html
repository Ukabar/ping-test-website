FAQ

What does Ping Test measure?

Ping Test measures HTTP-based latency, jitter, download speed, packet loss estimation, and a gaming score that summarizes current network quality.

Why is game latency different?

Actual game latency can differ because games use their own servers, protocols, routing paths, regions, and matchmaking infrastructure. Ping Test provides HTTP-based estimates, not direct game server measurements.

Does the app require permissions?

The app does not request unnecessary permissions. It only uses capabilities needed to run network tests, save local preferences, and display app features.

How can I report a bug?

Email hornet.devlo@gmail.com with your device model, iOS version, app version, what happened, and steps to reproduce the issue.
